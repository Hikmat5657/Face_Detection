# Project details
this is a pyhton project that aims to regonizes faces using machine learning model. 
In order to detect face we have to follow these steps:
1. prepare image 
2. load image to code
3. apply image to the model
4. model will output a rectangle to highlight where the face is.
5. draw rectangle on top of the image
6. finally, displays the final image with the rectangle overlay.
